Tools

Use this button to enable the eraser. To erase an object, simply click or tap the object. Note that any objects that the target object is over will be erased as well.
Use this button to place text or text boxes on your map. After you press the button, use the dialog to type in your text or use radio buttons to select preset text to use or modify. You can also toggle the background, and set the font and background colors. Upon clicking submit, the text object will appear on the map and can be dragged to a chosen location. Note that text objects can be deleted like other objects.
Use this button to enable rotation of objects. Tapping or clicking a classroom object in rotate mode will rotate it 45 degrees clockwise. Custom images, rectangles, and other polygons can also be rotated via green anchors at the corners, which show up in rotate mode.
Use this button to add images that already exist online. Paste in a valid image URL and you will add an image that can be rotated and resized via anchors. Images with width or height greater than 400px will be resized when they are loaded.
Use this function to add polygons, rectangles, and circles.
Use this button to resize objects. Clicking on an object in resize mode will make it resizeable. Swiping up on the map will enlarge the object, and swiping down will reduce its size. Circles, rectangles, other polygons, and custom images can be resized via grey circular anchors that show up in resize mode.

Top Buttons

Within Map Settings, there are four aspects of the map you can modify.
Dimensions: Use this button to change the width and height of your classroom map.
Colors: Use this button to open a dialog that allows you to set the colors of the floor and of each wall.
Show/Hide Grid: Use this button to toggle the grid on and off. The grid can be used to align objects and to better visualize out how classroom space is divided.
Show/Hide Walls: Use this button to toggle the walls on and off.
You also have these two features at the top nav bar.
Clear All: Use this button to erase all items from the map.
Capture: Use this button to capture an image of your map. Before clicking the button, make sure pop-ups are enabled on your web browser.
Once you have your image, right-click the image (you can also command-click on Macs / control-click on PC's) and select 'save' if you are on a computer.
If you are on an iPad, capture will still work, but you'll need an extra step. After capturing, press the home button and on/off button simultaneously to take a screen capture that saves to your camera roll.
